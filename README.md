# PREDDI

**Tu centro de control inmobiliario.**

Marketplace inmobiliario y plataforma de administración de propiedades.

## Módulos

- **Marketplace** — Publica, busca y conecta con propiedades en venta o alquiler.
- **Administración de Propiedades** — Gestiona contratos, cobros, mantenimiento e inquilinos.
- **Remates Judiciales** — Accede a oportunidades de inversión en remates judiciales.

## Stack

- [Next.js](https://nextjs.org) 16 (App Router)
- TypeScript
- Tailwind CSS 4

## Desarrollo

```bash
pnpm install
pnpm dev
```

Abre [http://localhost:3000](http://localhost:3000) en tu navegador.

## Deploy

Desplegado en [Vercel](https://vercel.com) con dominio personalizado: [preddi.app](https://preddi.app)
